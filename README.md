## Project Name
Smart Agriculture Drone using ESP32

## Project Scope
This project uses ESP32, DHT22, HC-SR04, Servo Motor, and a Potentiometer to simulate a smart agriculture drone. The system monitors temperature, humidity, soil moisture, and crop distance. When the soil is dry and crops are nearby, the servo motor activates the spraying mechanism automatically.
